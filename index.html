Usage

Firstly, include the ns.hover.min.js just before the end body tag in your html

<script src="src/ns.hover.min.js"></script>

just call with the container class and set the options to customize the look and feel. (You must have to set a container class in order to make the child elemnts use this effect. child can be any html tags including images offcourse.)

<script>
$(container_class_or_id).nsHover({
    scaling : false,
    speed: 'normal',
    rounded: 'normal',
    bgcolor: '#ffffff',
    bgopacity : 0.5,            
    bgpic : 'imgs/lens.png',
    bgsize : '50%',
    bganim : 'fade',
    border : 'none',
    content: ''
});
</script>

Options

scaling

set this value to anything from 0.0 - 1.0 to make the element shrink on hover, or set it 0 / false to disable scaling effect

Default Value:

false

Possible Values:

false / 0 / 0.0 - 1.0

speed

this options set the speed of the hover transitions and animation effects

Default Value:

normal

Possible Values:

normal / fast / slow

rounded

Set this option to control the border radius of the element. you can set predefined values like circle or set any custom values you want.

Default Value:

normal

Possible Values:

none(0%) / normal(5%) / circle(50%) / xx% / xx px

bganim

The background animation style of the overlay layer. Currently it is limited to Fade, Scale and Slide(from up) effecrs

Default Value:

fade

Possible Values:

fade / scale / slide

bgcolor

The background color of the overlay layer.

Default Value:

normal

Possible Values:

none(0%) / normal(5%) / circle(50%) / xx% / xx px

bgopacity

The background opacity of the overlay layer.

Default Value:

0.5

Possible Values:

0.0-1.0

bgpic

The background image of the overlay layer.

Default Value:

imgs/lens.png

Possible Values:

any image url

bgsize

The background image size of the overlay layer. It is centered by default

Default Value:

25%

Possible Values:

0% - 100%

shadow

The box shadow of the main block. Enable / Disable it by setting true or false

Default Value:

false

Possible Values:

true / false

content

If you want to show some text / icon / image on hover we have limited support for this also. you can set any valid html here, but be careful not to put much more content here or it will not look good. example : "click to buy", "know more" etc. you can style this content by external styles using classes

Default Value:

''

Possible Values:

any valid html content
